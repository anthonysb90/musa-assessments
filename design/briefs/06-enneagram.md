# Brief — Enneagram

1. **Positioning.** The believer's core type turned toward Christ, with gift, trap, and a written devotion. Read reflectively, more than once.
2. **Archetype.** Narrative-led profile.
3. **Signature visual.** The **Enneagram nonagon** drawn true — nine points on a circle, the core type node lit, the top-3 wings/connections traced in indigo. No other report uses the nine-point figure.
4. **Cover.** The nonagon in fine indigo line on a point-lattice field, the type node a filled gold dot.
5. **Rhythm.** Nonagon hero → core type narrative → gift / watch-out / where-you-grow → a full **written devotion** spread → Scripture. Medium-long, contemplative.
6. **Accent / texture.** Indigo `#3B5B7A` / geometric point-lattice.
7. **Bespoke.** Prayer/devotion card; Scripture; the ranked-9 with top-3. (No character card, no gap.)
8. **Target pages.** Medium-long — 7–9 pages.
9. **Must NOT look like.** Kingdom Design (both typologies with devotion). Kept apart: Enneagram = *nonagon*, indigo, one type deep; KDP = *dichotomy spectra + 16-grid*, navy/gold, composite.
